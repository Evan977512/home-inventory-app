# HOME nVentory 
- This web application is built for managing home inventory.

## 🌱 Features 
- Save Username, Encrypted Password, Reference of Profile Picture, and Email in <strong>Mongo DB</strong>. 

## ✏️ Core Features
- REST API

## 📌 Tech Stack
- MongoDB
- Express
- Node.js
- React.js
